\# Second Brain App



\## Description

A simple AI-powered knowledge management system where users can save notes and retrieve them later.



\## Tech Stack

\- Frontend: HTML, JavaScript

\- Backend: FastAPI (Python)

\- Database: MongoDB

\- AI: OpenAI API



\## Features

\- Add notes

\- View saved notes

\- Backend API with Swagger documentation

\- AI-ready architecture



\## How to Run

1\. Start MongoDB

2\. Run backend:

&nbsp;  uvicorn main:app --reload

3\. Open index.html in browser
